## New language features in Kaleidoscope

### Digit separator
Use `'` (single quote) and `_` (underscore) as digit separator

```
0x80'00'00'00
100'0000'0000UL
123_456_789
3.14_15_926f
0xFFFF'FFFF_FFFF'FFFF (Yes you can mix them!)
```
### Binary number literal
```
0b10010011
0b10'11'01'00
0b1100_1100_1100
```

### Octal number literal
```
0o103752
0o777'666'555
```
